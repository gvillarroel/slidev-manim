# Radial Fan-Out Focus

Use this pattern when one red hub should stay responsible for several related blocks while attention moves between branches.

## Core read

- Keep the hub visibly alive while branches leave it. The viewer should read the center as the cause, not as a leftover ornament.
- Reveal one dominant spoke first, then stage the other cards after it. Simultaneous fan-out turns into inventory instead of authored growth.
- Let only one branch own detailed associated chips at a time. Secondary detail should confirm focus, not flatten the scene into five equally loud regions.
- During a handoff or rotation beat, move the focus frame and local chips as one local cluster. If the branch rotates but the associated detail feels detached, the system stops reading as one object.
- Do not let one solid foreground block sit on top of another solid block. Background traces can pass behind the active card, but chips, callouts, and focus indicators need visible clearance from the card body.
- Prefer open-corner focus marks and card-edge chip anchors over closed highlight rectangles when the focused branch is dense. This preserves the feeling of precision without turning the active card into a stacked box collision.
- At phase boundaries, the next phase must reconstruct the prior phase's terminal positions for any actor that fades out. Opacity easing alone still reads as a cut if chips, focus marks, or beams snap back to default coordinates before disappearing.
- Keep the red hub fixed when it is the narrative anchor. If a rotated chip cluster pulls the scene off-center, compact or relocate the peripheral details inside the active branch instead of moving the hub to satisfy framing.
- Keep orbit turns moderate when the branch has attached detail. Large rotations can be technically centered around the hub while still making the active card and chips feel like they are falling out of the frame.
- Preserve destination scaffold opacity across the opening-to-fan-out boundary. If the fan-out reset hides prepared slots just as the red ingress trail becomes dominant, composition audits and viewers both read the proof frame as left-heavy.
- In resolved holds, retire or soften the local chip cloud and keep the main cards plus hub as the lasting structure.

## Proof frames to sample

- First complete fan-out with all primary cards visible and the hub still strongest.
- Active focus frame with one card plus its associated chips.
- Rotation or handoff frame where the whole system still reads as one orbiting object.
- Final hold after the chips have quieted and the main branch layout stays balanced.
