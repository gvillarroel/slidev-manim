# Quality Negative Space Focus

## Hypothesis

If the stage uses deliberate negative space and only one active zone at a time, the result will feel more premium than a composition that tries to occupy the whole frame constantly.

## Purpose

This spike tests whether a sparse composition with a drifting focus area can look more intentional than a fully populated diagram-like layout.

## Run the render

From the repository root:

```bash
uv run --script spikes/quality-negative-space-focus/main.py
```

This writes the rendered assets to:

```text
videos/quality-negative-space-focus/
```
