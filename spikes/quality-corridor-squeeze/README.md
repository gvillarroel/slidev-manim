# Purpose

Test whether forcing forms through a narrow corridor creates a stronger sense of authored compression than a normal approach.

# Hypothesis

If the forms pass through a constrained corridor before opening into the landing cluster, the motion should feel tighter and more intentional than a normal transfer.

# Run

```bash
uv run --script spikes/quality-corridor-squeeze/main.py
```
