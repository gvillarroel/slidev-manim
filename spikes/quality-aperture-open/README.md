# Purpose

Test whether a destination that appears through an opening aperture feels more designed than a normal reveal into open space.

# Hypothesis

If a pair of neutral shutters opens and then disappears before the landing frame, the reveal should feel more authored without depending on text or heavy ornament.

# Run

```bash
uv run --script spikes/quality-aperture-open/main.py
```
