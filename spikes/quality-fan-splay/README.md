# Purpose

Test whether a landing that opens into an angled fan feels more designed than a normal compact cluster.

# Hypothesis

If forms arrive through a narrow approach and then splay into a controlled angled arrangement, the final frame should feel more premium than a simple stacked destination.

```bash
uv run --script spikes/quality-fan-splay/main.py
```
