# Purpose

Test whether a short recoil after a decisive arrival makes a landing feel more authored than a simple settle.

# Hypothesis

If the dominant form snaps into place and recoils slightly before stabilizing, the motion should feel sharper and more intentional than a smooth arrival.

# Run

```bash
uv run --script spikes/quality-snap-recoil/main.py
```
