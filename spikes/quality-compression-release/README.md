# Quality Compression Release

## Hypothesis

If a scene compresses into a tight cluster before releasing into a resolved arrangement, the motion will feel more intentional than a single steady transform.

## Purpose

This spike tests whether pressure-and-release improves the perceived quality of a simple shape-driven sequence.

## Run the render

From the repository root:

```bash
uv run --script spikes/quality-compression-release/main.py
```

This writes the rendered assets to:

```text
videos/quality-compression-release/
```
