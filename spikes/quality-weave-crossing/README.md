# Purpose

Test whether a controlled crossing of paths feels more authored than parallel movement when the hierarchy remains clear.

# Hypothesis

If the dominant form crosses in front of the supporting forms through a visible weave moment, the transfer should feel more designed than a flat lateral move.

# Run

```bash
uv run --script spikes/quality-weave-crossing/main.py
```
