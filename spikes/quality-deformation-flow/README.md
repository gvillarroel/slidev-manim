# Quality Deformation Flow

## Hypothesis

If a video relies on shape deformation, spacing, and color continuity instead of explanatory text, the motion will feel more premium and will stay readable once narration is added later.

## Purpose

This spike tests a shape-first flow where three colored capsules become a vertical handoff stack. The video uses almost no text and depends on deformation, pacing, and accent motion to communicate the transition.

## Run the render

From the repository root:

```bash
uv run --script spikes/quality-deformation-flow/main.py
```

This writes the rendered assets to:

```text
videos/quality-deformation-flow/
```
