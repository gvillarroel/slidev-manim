# Purpose

Test whether a visible funnel geometry can make multiple inputs collapsing into one outlet feel authored instead of looking like a generic regroup.

# Hypothesis

If several forms compress through a narrowing funnel before resolving into one dominant landing, the merge should read as one deliberate convergence event rather than a loose cluster.
