# Purpose

Test whether a narrow throat between two gate bars makes a transfer feel more authored than a normal compressed passage.

# Hypothesis

If the dominant form visibly squeezes through a narrow throat before the final resolve, the transfer should feel more deliberate than a free passage through open space.
