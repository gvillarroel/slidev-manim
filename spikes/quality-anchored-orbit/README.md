# Quality Anchored Orbit

## Hypothesis

If supporting elements orbit around an anchor instead of moving linearly, the scene can feel more designed and less like a simple rearrangement.

## Purpose

This spike tests whether an anchored center with orbiting satellites creates more premium motion language than direct path swaps.

## Run the render

From the repository root:

```bash
uv run --script spikes/quality-anchored-orbit/main.py
```

This writes the rendered assets to:

```text
videos/quality-anchored-orbit/
```
