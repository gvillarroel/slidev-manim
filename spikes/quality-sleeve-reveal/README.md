# Purpose

Test whether one form sliding through a sleeve-like cover can make a reveal feel more authored than a plain slide-in.

# Hypothesis

If a sleeve stays visible while the dominant form passes through it and emerges cleaner on the far side, the reveal should read as a controlled uncovering rather than a generic move.

# Run

```bash
uv run --script spikes/quality-sleeve-reveal/main.py
```
