# Quality Edge Tension

## Hypothesis

If the main action pushes close to an edge without clipping awkwardly, the frame can feel more dynamic than a fully centered composition.

## Purpose

This spike tests whether controlled edge tension adds energy while still respecting the palette and overall balance.

## Run the render

From the repository root:

```bash
uv run --script spikes/quality-edge-tension/main.py
```

This writes the rendered assets to:

```text
videos/quality-edge-tension/
```
