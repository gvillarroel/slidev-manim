# Purpose

Test whether a visible bumper that redirects the dominant form makes the transfer feel more authored than a simple change in direction.

# Hypothesis

If the dominant form visibly compresses against a bumper before redirecting upward, the motion should feel more intentional than a free curved turn.

```bash
uv run --script spikes/quality-bumper-deflect/main.py
```
