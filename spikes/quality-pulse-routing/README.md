# Quality Pulse Routing

## Hypothesis

If one moving accent carries the viewer through the composition and the structural shapes stay calm, the video will feel higher quality than a layout where everything moves at the same time.

## Purpose

This spike tests a routing composition with almost no text. The scene uses colored stations, a single warm pulse, and staged emphasis instead of labels or dense annotation.

## Run the render

From the repository root:

```bash
uv run --script spikes/quality-pulse-routing/main.py
```

This writes the rendered assets to:

```text
videos/quality-pulse-routing/
```
