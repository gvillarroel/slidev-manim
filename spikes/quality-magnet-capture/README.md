# Purpose

Test whether a receiver that visibly pulls a form inward feels more authored than a destination that merely waits for the form to arrive.

# Hypothesis

If the receiving shape creates a visible capture pocket and the dominant form stretches into it before settling, the motion should feel like attraction rather than simple translation.
