# Quality Layered Reveal

## Hypothesis

If a video uses depth, masked reveals, and asymmetry instead of centered static layouts, the result will feel more intentional and less like a generic diagram export.

## Purpose

This spike tests a layered reveal with calm structure and one deliberate transformation moment. The composition stays sparse and depends on shape relationships more than verbal explanation.

## Run the render

From the repository root:

```bash
uv run --script spikes/quality-layered-reveal/main.py
```

This writes the rendered assets to:

```text
videos/quality-layered-reveal/
```
