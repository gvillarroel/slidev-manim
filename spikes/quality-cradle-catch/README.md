# Purpose

Test whether support forms catching the dominant form from below makes the landing feel more authored than a simple regroup.

# Hypothesis

If the support forms rise into a shallow cradle and the dominant form visibly lands into that catch before the final resolve, the sequence should feel more designed than a normal arrival.
